# 2.可靠的全局定位
# 3.无人工干预的自主路径规划与导航

## 核心功能
1. **多传感器数据采集、同步与统一接口管理**  
   - 封装激光雷达、IMU、GNSS和轮式编码器的数据采集 
   - 实现多传感器时间同步（10ms容差） 
   - 提供线程安全的数据队列 
   - 包含传感器配置管理和模拟器（用于测试）
2. **实时估计机器人运动位姿**
   - IMU预积分补偿激光雷达运动畸变 
   - 从点云提取几何特征（地面/平面/边缘点） 
   - 扫描匹配（ICP）估计相对运动 
   - 融合IMU、激光雷达和GNSS数据 
   - 输出高频位姿估计（前端里程计）
3. **全局地图优化与闭环检测**
   - 构建因子图优化问题（节点=位姿，因子=约束） 
   - 融合多种约束：里程计因子、GNSS因子、闭环因子 
   - 检测回环并消除累积误差 
   - 子图管理优化大范围建图 
   - 执行全局优化（类似g2o/GTSAM的功能）
4. **重定位与全局位姿估计**
   - 从先验地图中定位当前扫描 
   - 支持多种定位方式：GNSS提示、扫描匹配、扫描上下文 
   - 处理"绑架问题"（全局重定位） 
   - 提供定位置信度评估 
   - 构建和维护全局特征描述子数据库
5. **路径规划与运动控制**
   - 占据栅格地图构建与更新 
   - A*全局路径规划（考虑障碍物膨胀） 
   - 动态窗口局部路径规划（DWA）与避障 
   - 碰撞检测与紧急停止 
   - 路径平滑与速度控制
# SLAM导航系统
 - 实时SLAM建图 - 使用模拟激光雷达数据进行实时地图构建与定位 
 - 传感器模拟 - 完整模拟激光雷达（Lidar）和IMU传感器数据流 
 - 多模式可视化 - 通过RViz2提供实时建图、机器人位姿、粒子云等可视化 
 - 系统健康测试 - 完整的系统自检与性能基准测试套件 
 - 交互式菜单 - 用户友好的终端菜单系统，一键启动各项功能 
 - 完整系统集成 - 支持多节点协同工作，模拟完整SLAM导航流程
